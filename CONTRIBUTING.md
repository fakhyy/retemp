# Contributing

## Setup

Clone the repo and install dependencies:

```sh
git clone https://github.com/fakhyy/retemp
cd retemp
bun install
```

## Development

Build the library in watch mode:

```sh
bun run dev
```

## Testing

Tests use Bun's built-in test runner.

```sh
bun test
```

## Building

```sh
bun run build
```

## Pull Requests

1. Ensure all existing tests pass (`bun test`).
2. Add tests for any new functionality.
3. Keep changes focused — one feature/fix per PR.
